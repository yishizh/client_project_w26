# UMSI Career Development Resource Hub

A responsive website designed for the University of Michigan School of Information Career Development Office to help students access career resources and guidance.

## Pages Included
- **Home** (index.html) - Main landing page with overview and popular resources
- **Resumes** (resume.html) - Resume writing guides, templates, and tips
- **Networking & Connections** (networking.html) - Networking strategies and resources
- **Job Search** (job-search.html) - Job search strategies and UMSI resources

## Project Purpose
This website serves as a centralized hub for UMSI students to find and utilize career development resources, replacing scattered information with an organized, accessible, and user-friendly platform.

## Features
- Fully semantic HTML5 structure
- Accessible design (WAVE and aXe compliant)
- Responsive navigation across all pages
- Integration with UMSI Career Development Office resources
